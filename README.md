# Bank-analytics-project
##  Project Overview
Analyzed banking data to identify trends in customer behavior, revenue, and loan performance.

##  Tools Used
- Advanced Excel
- SQL
- Power BI
- Tableau

##  Key Tasks Performed
- Data cleaning and preprocessing
- KPI calculation and analysis
- Dashboard creation
- Insights and recommendations

##  Outcome
Created interactive dashboards to support data-driven decision making.
